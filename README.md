# ByAndroid

> 能让开发者直接拿过去依赖，迅速开发，也就是所谓的“Android技术中台”


## Open-source libraries

 - AndroidX
 - MVP - 项目整体架构
 - DataBinding - 控件id绑定
 - Retrofit2 & OkHttp3 - Construct the REST APIs.
 - Glide - Loading images from network.
 - immersionbar - 状态栏
 - RxBus - 事件总线
 - ByRecyclerView - 列表 上拉加载
 - ByWebView - WebView封装
